Minecraft Java版 操作方法①

📅 2023年6月30日 ✍️ akitaclark0731-01@gmail.com
MinecraftMinecraft Java
Minecraft Java版のの基本的な操作を紹介します！※Ver 1.20.1での場合です。 ステップ1 歩こう！ Wキーで前進、Sキーで後進、Aキーで左、Dで右へ歩きます！ ステップ2 ジャンプしよう！ スペースキーでジャンプができます！Wキーを押しながらタイミングよくスペースキーを押すと段差を越えることができます！ ステップ3 走ろう！ Wキー＋Ctrlキーで走ることができます！ ステップ4 ブロックを掘ろう！ 左クリック長押しでブロックを掘ることができます！ほとんどのブロックは掘るとアイテム化します！ ステップ5 ブロックを置こう！ 今回は以上です！続きはMinecraft Java版 操作方法②で！
